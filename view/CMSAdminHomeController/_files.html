The File modules allow you to upload images and documents to the server for use on the website. Prerequisites for uploading files are:

  • Images must be in JPEG, GIF or PNG formats.
  • Documents must be less than 8MB in size.
  • We recommend documents are distributed in PDF format but the system allows Word Documents, Spreadsheets and PowerPoint presentations to be uploaded.

List View

The default page when clicking the Files tab is a thumbnail list view. At the top of the page is a Filter, typing text in this input box filters the thumbnail list below, for example: typing 'ne' will list all items containing the characters 'ne'. Alongside the 'Filter' is an 'Order By' select box, select a field to reorder the list below. Below the the filter is a list of File items, under each item are the options to 'Edit' - the filename and 'Delete' - the trash can. Clicking the filename will take you to the Edit Form, clicking the trash can icon will ask you to confirm before removing the item.

Thumbnail views of images are generated automatically from the original files, if a thumbnail cannot be created a default document image is used.

Create/Edit View

Selecting the 'Create New Site Section' or the 'Edit' button next to a list item takes you to the same form.

  • Upload File: click the 'Browse' button to select a file from your computer.
  • Caption: Used as alternative text for images or documents in cases where they cannot be displayed as icons or images.
Uploading a file can take anywhere between 10 seconds - 2 minutes depending on your connection speeds and filesize.
