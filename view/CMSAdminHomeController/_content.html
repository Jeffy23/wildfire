Probably the most used module in the CMS, all CMS controlled content is added here.

List View

The default page when clicking the Content Sections tab is a list view. At the top of the page is a Filter, typing text in this input box filters the list below, for example: typing 'ne' will list all items containing the characters 'ne'. Alongside the 'Filter' is an 'Order By' select box, select a field to reorder the list below. Below the filter box is an select box which, when selected, lists content items by there assigned sections, the list will update on selection. Below the the section select box is a list of Content items, to the right of each item are the options to 'Edit', 'Delete' and 'View'. The Edit button will take you to the Edit Form, clicking the 'Delete' option will ask you to confirm before removing the item. The View button will take you through to a view of the content on the public site.

Create/Edit View

Selecting the 'Create New Site Content' or the 'Edit' button next to a list item takes you to the same form, although the 'Edit' WYSIWYG editor gives you two additional options.

  • Title: Content title, the url for the item is auto generated from this title e.g. 'My New News Item' will have the url '/my-new-news-item'.
  • Choose a Section: Assign a content item to a section of the site, for example 'News'.
  • Status: Draft - item will not appear on the site, or Published - item will appear on the site.
  • Date to be Published: This sets a future date for an item to be published, 'Status' must also be set to published for this to work.
  • Content: What You See Is What You Get (WYSIWYG) editor. This allow basic text formatting. See the following list.

    • Bold

      To add formatting - Highlight desired text, click the 'Bold' icon.

      To remove formatting - Highlight desired bold text, click the 'Bold' icon.

    • Italic

      To add formatting - Highlight desired text, click the 'Italic' icon.

      To remove formatting - Highlight desired italic text, click the 'Italic' icon.

    • Link

      To add formatting - Highlight desired text, click the 'Link' icon. A popup will appear:

      • Option 1, if you want to enter a link to another website select the 'click here', this will guide you through a series of windows allowing you to enter a url and title for the link.
      • Option 2, (only available if documents have been uploaded via the Files module) this allows you to link through to an item by selecting it from the pulldown menu.

      To remove formatting - Highlight desired link text, click the 'Link' icon.

    • Bullet List

      To add formatting - Highlight desired text - normally over multiple lines, click the 'Bullet List' icon.

      To remove formatting - Highlight desired list text - normally over multiple lines, click the 'Bullet List' icon.

    • Numbered List

      To add formatting - Highlight desired text - normally over multiple lines, click the 'Numbered List' icon.

      To remove formatting - Highlight desired list text - normally over multiple lines, click the 'Numbered List' icon.

    • Change Block Text, predefined text formatting for headings, sub-headings and quotes.

      To add formatting - Highlight desired text - select the relevant option from the pulldown list.

      To remove formatting - Highlight desired list text - select the 'paragraph' option form the pulldown list.

    • Add Image - ONLY AVAILABLE ONCE AN ITEM HAS BEEN SAVED

      To add an image - Click the 'Add Image' icon, a popup appears. The popup lists all images loaded via the 'Files' module. Simply drag the image from the list to one of the empty place holders on the right-hand side. The application will decide how the image is displayed in relation to the content on the public site.

      To remove an image - Click the 'Add Image' icon, a popup appears. Simply drag it from the right-hand side place holder back to the list on the left.

    • Select Category - ONLY AVAILABLE ONCE AN ITEM HAS BEEN SAVED

      To assign content to a category - Click the 'Select Category' icon, a popup appears. The popup lists all categories loaded via the 'Category' module. Select the relevant categories using the checkboxes next to each category title and click 'Update Categories'.

      To remove relationship to a category - Click the 'Select Category' icon, a popup appears. Uncheck the box next to each of the selected categories and click 'Update Categories'.
